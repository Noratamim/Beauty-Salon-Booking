 #  Beauty Salon Booking System (MVP)

## Project Description
This project is a web-based **Minimum Viable Product (MVP)** for a beauty salon booking system. It allows customers to browse services and book appointments online, while administrators can manage bookings through a dedicated dashboard. The system features a modern **Glassmorphism UI** design.

## Features
- **User Interface:** Responsive "Glassmorphism" design with a fixed scenic background.
- **Booking System:** Customers can select services (Hair, Makeup, etc.) and book specific dates.
- **Validation:** Prevents booking appointments in the past.
- **Admin Dashboard:** View all appointments and **delete** canceled ones (CRUD operations).
- **Database Integration:** Full MySQL integration for storing services and bookings.

## Technologies Used
- **Frontend:** HTML5, CSS3, Bootstrap 5.
- **Backend:** PHP.
- **Database:** MySQL.
- **Tools:** VS Code, XAMPP.

##  Setup Steps (How to Run)
1. Install **XAMPP** on your machine.
2. Start **Apache** and **MySQL** from the XAMPP Control Panel.
3. Copy the project folder to `C:\xampp\htdocs\salon`.
4. Open your browser and go to `http://localhost/phpmyadmin`.
5. Create a new database named `salon_db`.
6. Import the `salon_db.sql` file (found in this repository) into the database.
7. Open your browser and visit: `http://localhost/salon`
